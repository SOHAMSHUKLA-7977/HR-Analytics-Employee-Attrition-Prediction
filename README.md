# HR Analytics – Employee Attrition Prediction

## Overview

This project analyzes the IBM HR Analytics Employee Attrition dataset to identify the factors influencing employee turnover and predict employee attrition using Machine Learning.

## Objectives

- Perform Exploratory Data Analysis (EDA)
- Identify factors affecting employee attrition
- Build a Random Forest Classification model
- Evaluate model performance
- Visualize feature importance

## Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset

IBM HR Analytics Employee Attrition Dataset

## Project Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis
- Label Encoding
- Train-Test Split
- Random Forest Classification
- Model Evaluation
- Feature Importance Analysis

## Results

The Random Forest model successfully predicts employee attrition and highlights key factors such as Overtime, Monthly Income, Job Satisfaction, Age, and Work-Life Balance that influence employee retention.
